# JIS Z 8210 pictograms (標準案内用図記号)

Vendored map-marker pictograms for the categories that appear on physical
station signage: toilets (5106), men (5107), women (5108), accessible
facility (5109), elevator (5127), escalator (5128), stairs (5131).

- Source: [cat-in-136/JISZ8210_Symbols_SVG](https://github.com/cat-in-136/JISZ8210_Symbols_SVG),
  traced from the ECOMO Foundation's official vector pamphlet
  (ecomo.or.jp 標準案内用図記号ガイドライン). The symbols are the JIS Z 8210
  standardized public-information symbols, published for free use per the
  ECOMO guideline (do not distort or restyle the figures).
- Local processing: background plate removed, figure paths set to
  `fill="currentColor"`, knockout paths set to `fill="var(--marker-bg)"`,
  sized 14×14 for the 24px marker bubbles. The figure geometry is unchanged.
